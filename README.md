#Finance Transaction Classifier

This project builds an end-to-end Machine Learning system that automatically categorizes financial transactions (Food, Rent, Travel, Utilities, Entertainment). It includes data cleaning, fuzzy matching, feature engineering, model training, and a Streamlit demo app.
